FormLabs Form1

  • Stereo Lithography
  • Adjustable (high) resolution
  • Smart printer with memory
  • Not good with solids/closed regions
  • Slow (10-20hr prints)
  • Expensive
  • Smart printing software (support structures)

PrintrBot Simple Maker's Kit

  • Extrusion
  • Low-ish resolution
  • No onboard memory
  • Fast (1-3hr prints)
  • Cheap
  • Smart fill
  • Free printing software (Repetier/Slicr)
